Update default browser in manjaro i3 arch linux

Jan 1, 2021
1 min read

By default Manjaro i3 has palemoon installed as the default browser. I’m not a big fan, and want to switch over to something that is closer to my heart, namely opera.

These are the steps that you need to do:

  • Remove all entries from the ~/.config/mimeapps.list
  • Update the .profile file

Update mimeapps

vim ~/.config/mimeapps.list

To replace all the occurances of userapp-Pale Moon.desktop with opera.desktop you need to enter(first press Esc key and :)

%s/userapp-Pale Moon.desktop/opera.desktop/g

Update .profile

Next edit the ~/.profile file and change the BROWSER environment variable to /usr/bin/opera.

Done.
